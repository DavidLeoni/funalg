## Install

Requires Python 3.12.3+

1. Create virtual environment

```bash
python3 -m venv .venv
```

Activate environment: 

Linux:

```bash
source .venv/bin/activate
```


2. Install dependencies

```bash
python -m pip install -r requirements.txt
```

## Test

Run

```bash
python -m pytest -v -s test/
```
